# FlowRead — Read at the speed of thought

A browser-based speed reader using **Rapid Serial Visual Presentation (RSVP)** — a technique that eliminates eye movement, the biggest bottleneck in reading speed. Drop a PDF, EPUB, or paste text and read one focused word at a time.

---

## Features

- PDF, EPUB, and TXT support with drag-and-drop
- Focus letter highlighting at the optimal fixation point per word
- Adaptive speed — auto-adjusts WPM based on your reading behaviour
- Context words — faded prev/next word for reading flow
- Session stats — avg WPM, time taken, and word count on completion
- Full book sidebar with click-to-jump anywhere in the text
- Keyboard controls: `Space` play/pause · `↑↓` WPM · `←→` skip
- Dark and light theme · fully responsive

---

## Tech

Vanilla JS, plain CSS, no framework, no build step.  
PDF parsing via PDF.js · EPUB parsing via JSZip.
